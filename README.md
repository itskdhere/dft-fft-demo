# dft-fft-demo

This is a simple demo program written in **C** for computing and benchmarking *Discrete Fourier Transform (DFT)* and *Fast Fourier Transform (FFT)* algorithms.

Compile:

```bash
gcc -o fourier_transforms fourier_transforms.c
```

Run:

```bash
./fourier_transforms
```

References:

- [Cooley, James W., and John W. Tukey, 1965, “An algorithm for the machine calculation of complex Fourier series,” Math. Comput. 19: 297-301](https://www.ams.org/journals/mcom/1965-19-090/S0025-5718-1965-0178586-1/S0025-5718-1965-0178586-1.pdf)
